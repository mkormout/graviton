**body.gd changes (per D-02, D-03):**

1. Add two signal declarations at the top of the class, after the class_name line and before the @export block:
```gdscript
signal died()
signal health_changed(old_health: int, new_health: int)
```

2. In `damage()` function, save the old health before mutation and emit the signal when health decreases. Replace the existing function body:
```gdscript
func damage(attack: Damage):
    if not can_die or not attack:
        return
    var total = attack.calculate(defense)
    var old_health := health
    health += total
    if health < old_health:
        health_changed.emit(old_health, health)
    if health <= 0:
        die()
```
IMPORTANT: Only emit `health_changed` when `health < old_health` (damage, not healing). This prevents wave multiplier reset on health pack pickup (Pitfall 6 from RESEARCH.md).

3. In `die()` function, add `died.emit()` on the line immediately before `queue_free()`:
```gdscript
    if item_dropper:
        item_dropper.drop()
    died.emit()
    queue_free()
```
CRITICAL: `died.emit()` MUST come before `queue_free()` so signal receivers still have a valid reference to the Body node (Pitfall 2 from RESEARCH.md).
