**5. Update _spawn_enemy to accept speed_tier** — Change the signature and body of `_spawn_enemy` (currently line 94):

From:
```gdscript
func _spawn_enemy(enemy_scene: PackedScene) -> void:
    var enemy := enemy_scene.instantiate()
    enemy.tree_exiting.connect(_on_enemy_tree_exiting)
    enemy.add_to_group("enemy")
    if ScoreManager:
        ScoreManager.register_enemy(enemy)
    get_parent().add_child(enemy)
    ...
```

To:
```gdscript
func _spawn_enemy(enemy_scene: PackedScene, speed_tier: float = 1.0) -> void:
    var enemy := enemy_scene.instantiate()

    # Set speed_tier BEFORE add_child so _ready() receives it
    if speed_tier != 1.0 and enemy.get("speed_tier") != null:
        enemy.speed_tier = speed_tier

    enemy.tree_exiting.connect(_on_enemy_tree_exiting)
    enemy.add_to_group("enemy")
    if ScoreManager:
        ScoreManager.register_enemy(enemy)
    get_parent().add_child(enemy)
    enemy.global_position = _get_spawn_position()
    get_parent().setup_spawn_parent(enemy)
```

IMPORTANT: `speed_tier` must be set BEFORE `get_parent().add_child(enemy)` because Swarmer reads `speed_tier` in `_ready()`, which fires on `add_child`.

**6. Update trigger_wave group loop** — In `trigger_wave()`, update the group iteration loop (currently lines 85-92) to pass speed_tier:

From:
```gdscript
for group in groups:
    var enemy_scene: PackedScene = group.get("enemy_scene")
```

To:
```gdscript
for group in groups:

    if not enemy_scene:
        push_warning("[WaveManager] Group in wave %d has no enemy_scene" % (_current_wave_index - 1))
        continue
    for i in range(count):
        _spawn_enemy(enemy_scene, speed_tier)
```
