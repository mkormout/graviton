**Create NEW file `prefabs/ui/controls-hint.gd`** with this exact content:
```gdscript
class_name ControlsHint
extends CanvasLayer

@onready var _panel_container: MarginContainer = $MarginContainer
@onready var _toggle_button: Button = $ToggleButton

var _visible_state: bool = false

func _ready() -> void:
	_panel_container.visible = false
	_toggle_button.pressed.connect(toggle)

func toggle() -> void:
	_visible_state = not _visible_state
	_panel_container.visible = _visible_state
	_toggle_button.text = "\u25C4" if _visible_state else "\u25BA"
```

Note: `\u25BA` = right-pointing triangle, `\u25C4` = left-pointing triangle. The MarginContainer is hidden by default (D-04); the ToggleButton remains visible at all times (D-06). The keyboard shortcut (TAB) is handled by world.gd, not this script — this script only handles the Button click.
