`_on_combo_expired() -> void` (per D-08):
- Set `_combo_value.text = "--"`
- Set `_combo_value.add_theme_color_override("font_color", Color(0.5, 0.5, 0.5))`
- No animation (per D-11)

`_animate_score_flash() -> void` (per D-10):
- Kill any running score tween first (Pitfall 1): `if _score_tween and _score_tween.is_running(): _score_tween.kill()`
- Reset to white: `_score_value.add_theme_color_override("font_color", Color.WHITE)`
- Create tween: `_score_tween = _score_value.create_tween()`
- Flash to light-yellow then back: `_score_tween.tween_property(_score_value, "theme_override_colors/font_color", Color(1.0, 1.0, 0.7), 0.1)`
- Chain back to white: `_score_tween.chain().tween_property(_score_value, "theme_override_colors/font_color", Color.WHITE, 0.2)`
- NOTE: If `tween_property` does not work with `theme_override_colors/font_color` path (Assumption A1), use fallback: replace tween lines with `_score_value.add_theme_color_override("font_color", Color(1.0, 1.0, 0.7))` and use `_score_tween.tween_callback(func(): _score_value.add_theme_color_override("font_color", Color.WHITE)).set_delay(0.15)`.

`_animate_multiplier_pulse() -> void` (per D-09):
- Kill any running mult tween first (Pitfall 1): `if _mult_tween and _mult_tween.is_running(): _mult_tween.kill(); _mult_value.scale = Vector2.ONE; _mult_value.add_theme_color_override("font_color", Color.WHITE)`
- Create tween: `_mult_tween = _mult_value.create_tween()`
- Set parallel mode: `_mult_tween.set_parallel(true)`
- Scale up: `_mult_tween.tween_property(_mult_value, "scale", Vector2(1.4, 1.4), 0.2)`
- Color to gold: `_mult_tween.tween_property(_mult_value, "theme_override_colors/font_color", Color(1.0, 0.843, 0.0), 0.2)`
- Chain scale back down: `_mult_tween.chain().set_parallel(true)`
- `_mult_tween.tween_property(_mult_value, "scale", Vector2.ONE, 0.2)`
- Color back to white: `_mult_tween.tween_property(_mult_value, "theme_override_colors/font_color", Color.WHITE, 0.2)`
- Total duration: ~0.4s (per D-09)
