**PART A: Beeliner, Sniper, Flanker, Swarmer** (all have existing CoinDropper):

These 4 scenes share the same modification pattern. For each:

1. Add a new ext_resource for the health-pack scene. Use the next available id number in each file. Example for Beeliner (currently has ids up to `5_ammo`):
   ```
   [ext_resource type="PackedScene" uid="{health-pack-uid}" path="res://prefabs/health-pack/health-pack.tscn" id="6_healthpack"]
   ```
   For Sniper/Flanker/Swarmer, similarly use the next available id (they also go up to `5_ammo`, so use `6_healthpack`).

2. Add a new sub_resource for the health-pack ItemDrop:
   ```
   [sub_resource type="Resource" id="Resource_health_drop"]
   script = ExtResource("3_itemdrop")
   model = ExtResource("6_healthpack")
   chance = 0.11
   ```

3. Update the CoinDropper node's `models` array to include both the existing coin drop and the new health-pack drop:
   ```
   models = Array[ExtResource("3_itemdrop")]([SubResource("Resource_coin_drop"), SubResource("Resource_health_drop")])
   ```

4. **Do NOT change drop_count** -- keep existing values (Beeliner=2, Sniper=2, Flanker=2, Swarmer=1). Per D-09, the 10% is per-roll.

5. **Do NOT modify the AmmoDropper node** -- it remains unchanged.

**Important:** Update `load_steps` in the scene header if present. Beeliner has `load_steps=16` -- adding 1 ext_resource and 1 sub_resource means incrementing to `load_steps=18`. Sniper, Flanker, Swarmer may not have load_steps (they don't -- their headers just say `format=3`), so no change needed for those.
