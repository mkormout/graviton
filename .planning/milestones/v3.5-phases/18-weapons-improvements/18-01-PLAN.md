<objective>
Fix the recoil bug that causes unintended ship spin when firing, and add a shake() method to BodyCamera so heavy weapons can trigger screen shake.

Purpose: Recoil is broken today — firing any weapon applies an off-center torque because the impulse position is passed as `sender.global_position / 100` (a near-zero world coordinate) instead of Vector2.ZERO. This causes ships to spin unpredictably. Fixing it to `apply_central_impulse` eliminates all unintended torque. Camera shake is added here as the foundation used by charge weapons in Wave 2.

Output: One-line fix in mountable-body.gd; new shake() method in body_camera.gd.
</objective>

```gdscript
# Called by heavy weapons (Gausscannon, RPG, GravityGun) on fire.
# magnitude: pixel offset of shake. duration: seconds of shake.
func shake(magnitude: float = 8.0, duration: float = 0.3) -> void:
    if _shake_tween and _shake_tween.is_running():
        _shake_tween.kill()
    _shake_tween = create_tween()

            Vector2(randf_range(-magnitude, magnitude),

            duration / 6.0)
    _shake_tween.tween_property(self, "offset", Vector2.ZERO, 0.05)
```
