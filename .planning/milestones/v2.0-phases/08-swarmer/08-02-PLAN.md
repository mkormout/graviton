**Key values and rationale:**

Root node export overrides:
- `max_speed = 1800.0` -- slightly lower than Flanker 2000.0 (Swarmers are slower individually)
- `thrust = 1200.0` -- lower than Flanker 1500.0 (slower acceleration, offset by numbers)
- `detection_radius = 10000.0` -- same as all other enemies
- `max_health = 15` -- lower than Beeliner 30 (D-15: low HP, dies in fewer hits). Beeliner has implicit 30 from EnemyShip base default; Flanker has 40; Sniper has 50. Swarmer at 15 is the weakest.
- `item_dropper = NodePath("CoinDropper")` -- CoinDropper wired to Body.item_dropper (called automatically on die); AmmoDropper called manually in die() override (established dual-dropper pattern)

CohesionArea (NEW -- D-06, D-14):
- `collision_layer = 0` -- area has no layer of its own
- `collision_mask = 1` -- detects bodies on Ship layer (Layer 1). Both Swarmers and PlayerShip are on layer 1; the `body is Swarmer` filter in swarmer.gd handles the distinction.
- `monitorable = false` -- only monitors, is not monitored (same as DetectionArea/HitBox pattern)
- `CircleShape2D_cohesion.radius = 700.0` -- matches the `cohesion_radius` export default. RESEARCH.md Pitfall 6: the shape radius MUST match the export default; otherwise cohesion never triggers.

FireTimer:
- `wait_time = 0.8` -- faster than Beeliner 1.5 but slower than Flanker 0.25. The Swarmer harasses with steady single shots (not bursts like Flanker, not salvos like Beeliner). This rate makes 5 Swarmers collectively produce ~6 shots/second -- chaotic pressure from multiple angles.

Barrel position:
- `position = Vector2(350, 0)` -- placed outside the collision shape radius (300) to avoid self-collision with HitBox. This follows the Flanker approach of using _barrel.global_position in _fire() (D-03). Barrel at 350 > HitBox radius 300 ensures bullets spawn clear of the hitbox.

Loot (D-16):
- CoinDropper: 1 copper coin at chance=1.0, drop_count=1 (fewer coins than Flanker/Sniper since Swarmers come in packs -- 5 Swarmers * 1 coin = 5 coins per wave, comparable to other enemy types)
- AmmoDropper: minigun-ammo at chance=0.4 / no-drop at chance=0.6, drop_count=1 (lower drop rate since many Swarmers die per wave)

No picker Area2D in scene (ENM-15).

**UID handling:** The executor should read the actual UID from `components/swarmer.gd.uid` if it exists (Godot 4.6.2 generates .uid files). If no .uid file exists, either omit the uid attribute on the ext_resource or let Godot assign one on first open. The uid values in ext_resources for item-dropper.gd, item-drop.gd, coin-copper.tscn, and minigun-ammo.tscn should be copied exactly from flanker.tscn (they reference the same shared resources).
