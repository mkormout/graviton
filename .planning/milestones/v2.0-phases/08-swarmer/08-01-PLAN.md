**_tick_state (per D-04, D-08, D-10, D-11, D-12):**

```gdscript
func _tick_state(_delta: float) -> void:
    if not is_instance_valid(_target):
        _target = null
        _change_state(State.IDLING)
        return

    var to_target: Vector2 = _target.global_position - global_position
    var dist: float = to_target.length()
    var force_scale: float = cohesion_thrust_scale if _nearby_swarmers.size() > 0 else 1.0

    match current_state:
        State.SEEKING:
            if dist <= fight_range:
                _change_state(State.FIGHTING)
            else:
                # Apply angle offset to steering direction (D-04)
                var raw_dir := to_target / dist
                var offset_dir := raw_dir.rotated(_angle_offset)
                apply_central_force(offset_dir * thrust * force_scale)
                rotation = linear_velocity.angle()
        State.FIGHTING:
            var target_angle := to_target.angle()
            rotation = lerp_angle(rotation, target_angle, 5.0 * _delta)
            apply_central_force((to_target / dist) * thrust * force_scale)
            # Hysteresis: 1.2x fight_range to prevent oscillation (D-12)
            if dist > fight_range * 1.2:
                _change_state(State.SEEKING)
```

IMPORTANT: Do NOT use `steer_toward()` -- it always applies full thrust with no scale parameter. Inline `apply_central_force()` to apply the cohesion force_scale (per D-07, D-08, and RESEARCH.md Pattern 3).

**_physics_process override (per D-08, RESEARCH.md Pattern 5):**

```gdscript
func _physics_process(delta: float) -> void:
    super(delta)
    if dying:
        return
    _apply_separation()
```

The `dying` guard AFTER `super(delta)` is critical -- `super(delta)` calls `EnemyShip._physics_process` which calls `_tick_state`, but does not return from the calling function. The explicit guard prevents separation from applying to dying Swarmers (Pitfall 4).

**_apply_separation (per D-07, RESEARCH.md Pattern 5):**

```gdscript
func _apply_separation() -> void:
    for swarmer in _nearby_swarmers:
        if not is_instance_valid(swarmer):
            continue
        var away: Vector2 = global_position - swarmer.global_position
        var dist: float = away.length()
        if dist < 1.0:
            continue  # Coincident -- skip to avoid NaN
        # Linear falloff: max force at dist=0, zero force at dist=cohesion_radius
        var strength: float = separation_force * (1.0 - clampf(dist / cohesion_radius, 0.0, 1.0))
        apply_central_force(away.normalized() * strength)
```

Use `is_instance_valid()` guard for every swarmer access -- dead Swarmers may remain in the array for 1 frame after `queue_free()` (Pitfall 1). Use linear falloff, NOT inverse-square (Pitfall: jitter at close range).
