**Class declaration:**

class_name Flanker
extends EnemyShip

**Export variables (all @export):**
- `fight_range: float = 4500.0` — distance threshold for LURKING->FIGHTING (D-09, D-14)
- `return_range: float = 7500.0` — distance threshold for FIGHTING->LURKING; MUST be > fight_range for hysteresis (D-05, D-06, D-15)
- `orbit_entry_range: float = 9500.0` — distance threshold for SEEKING->LURKING (D-13)
- `orbit_radius: float = 7000.0` — desired orbit distance from player; MUST be > fight_range (D-17, D-19)
- `orbit_correction_strength: float = 0.15` — proportional radial correction gain (D-17)
- `bullet_speed: float = 5500.0` — bullet velocity on fire

**Instance variables:**
- `var _target: Node2D = null`
- `var _bullet_scene := preload("res://prefabs/enemies/flanker/flanker-bullet.tscn")`

**@onready variables:**
- `@onready var _fire_timer: Timer = $FireTimer`
- `@onready var _ammo_dropper: ItemDropper = $AmmoDropper`
- `@onready var _barrel: Node2D = $Barrel`

**func _ready() -> void:**
Call `super()`. Apply per-instance randomization:
- `thrust *= randf_range(0.8, 1.2)`
- `max_speed *= randf_range(0.8, 1.2)`
- `orbit_radius *= randf_range(0.8, 1.3)` — per D-19
- `orbit_direction = 1.0 if randf() > 0.5 else -1.0` — per D-18
Connect signals:
- `_fire_timer.timeout.connect(_on_fire_timer_timeout)`
- `detection_area.body_exited.connect(_on_detection_area_body_exited)`

**func _on_detection_area_body_entered(body: Node2D) -> void:**
Override the base class handler. Guard: `if dying: return`. If `body is PlayerShip and current_state == State.IDLING`: set `_target = body`, call `_change_state(State.SEEKING)`. Per D-12.

**func _on_detection_area_body_exited(body: Node2D) -> void:**
Follow Sniper pattern exactly. Only clear `_target` if `body == _target and current_state == State.SEEKING`. Then set `_target = null` and `_change_state(State.IDLING)`. Do NOT clear target in LURKING or FIGHTING — `is_instance_valid(_target)` guard in `_tick_state` handles those states.

**func _tick_state(_delta: float) -> void:**
First check: `if not is_instance_valid(_target):` then set `_target = null` and `_change_state(State.IDLING)`, return.

Compute shared values at top of function:
```gdscript
var to_target: Vector2 = _target.global_position - global_position
var dist: float = to_target.length()
```

Then `match current_state:`:

**State.SEEKING:**
- `look_at(_target.global_position)`
- If `dist < orbit_entry_range`: call `_change_state(State.LURKING)` — per D-13
- Else: call `steer_toward(_target.global_position)`

**State.LURKING:**
- Guard: `if dist < 1.0: return` — degenerate case guard
- Compute `toward_norm: Vector2 = to_target / dist`
- Compute tangential force: `var tangential: Vector2 = toward_norm.orthogonal() * orbit_direction` — uses Godot's built-in `Vector2.orthogonal()` which returns `Vector2(y, -x)` per D-17
- Compute radial correction: `var radius_error: float = dist - orbit_radius` then `var radial: Vector2 = toward_norm * radius_error * orbit_correction_strength` — proportional P-controller per D-17
- Apply combined force: `apply_central_force((tangential + radial) * thrust)` — all movement via apply_central_force per ENM-03
- Face direction of travel: `look_at(global_position + tangential)` — NOT the player, so barrel faces orbit direction during LURKING (Research Pattern 2)
- State transition: if `dist < fight_range`: call `_change_state(State.FIGHTING)` — per D-09, D-14
- NO firing in LURKING — per D-20

**State.FIGHTING:**
- Guard: `if not is_instance_valid(_target):` then `_change_state(State.LURKING)`, return
- `look_at(_target.global_position)` — face player so bullets fire toward them
- `steer_toward(_target.global_position)` — close distance while fighting
- If `dist > return_range`: call `_change_state(State.LURKING)` — per D-05, D-15

**func _enter_state(new_state: State) -> void:**
Print: `print("[Flanker] _enter_state: %s" % State.keys()[new_state])`
If `new_state == State.FIGHTING`: call `_fire()` then `_fire_timer.start()` — fire immediately on entry per D-02, D-08. Same pattern as Beeliner.

**func _exit_state(old_state: State) -> void:**
If `old_state == State.FIGHTING`: call `_fire_timer.stop()` — per D-08. Same pattern as Beeliner.

**func _fire() -> void:**
Guard: `if dying: return`
Single bullet, no spread (D-01 — rapid salvo comes from short timer interval, not multiple bullets per call):
```gdscript
var bullet := _bullet_scene.instantiate() as RigidBody2D
var fire_dir := Vector2.from_angle(global_rotation)
bullet.rotation = global_rotation
bullet.linear_velocity = fire_dir * bullet_speed
if spawn_parent:
    spawn_parent.add_child(bullet)
    bullet.global_position = _barrel.global_position
else:
    push_warning("Flanker: spawn_parent not set")
```
Uses `_barrel.global_position` like Sniper (not hardcoded offset like Beeliner) — per D-04, ENM-05.

**func _on_fire_timer_timeout() -> void:**
Guard: `if dying or current_state != State.FIGHTING: return` — prevents misfire after state exit. Then call `_fire()`. Per D-02.

**func die(delay: float = 0.0) -> void:**
Guard: `if dying: return`. Stop `_fire_timer.stop()`. Call `_ammo_dropper.drop()`. Call `super(delay)`. Per D-24. Identical pattern to Beeliner and Sniper.

**What NOT to include:**
- No FLEEING state logic (D-07, D-16)
- No AimTimer (that's Sniper-specific; Flanker fires immediately)
- No SPREAD_ANGLES (that's Beeliner-specific; Flanker fires single bullets)
- No `_integrate_forces` override (inherited max_speed clamp from EnemyShip is sufficient)
- No direct `linear_velocity` assignment anywhere (ENM-03)
