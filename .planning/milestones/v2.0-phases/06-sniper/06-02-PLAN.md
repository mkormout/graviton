# Plan 06-02: Sniper Scene + WaveManager Integration + Playtest

## Goal
Create the Sniper inherited scene with all required nodes and tuned export values, wire it into the WaveManager wave array, and verify the full Sniper gameplay loop via human playtest.

## Context
Plan 06-01 created `sniper.gd` and `sniper-bullet.tscn`. This plan assembles the Sniper scene that inherits `base-enemy-ship.tscn`, overrides the script to `sniper.gd`, adds the two Timers (FireTimer + AimTimer) and two ItemDropper nodes (CoinDropper + AmmoDropper), and wires the Sniper into `world.gd` for WaveManager spawning. The Beeliner scene (`beeliner.tscn`) is the structural template.

<task id="06-02-T03" type="checkpoint:human-verify" gate="blocking">
<title>Playtest Sniper behavior</title>
<what-built>
Complete Sniper enemy: scene inheriting base-enemy-ship, sniper.gd script with three-band distance management, aim-up telegraph fire pattern, FLEEING behavior, and heavy slow bullet. Wired into WaveManager as wave 2.
</what-built>
<how-to-verify>
1. Open project in Godot editor. Verify no parse errors in `sniper.gd` (Script panel > Open sniper.gd).
2. Verify `sniper.tscn` loads without errors (open in Scene panel).
3. Verify `sniper-bullet.tscn` loads without errors.
4. Run the game (F5).
5. Press F once to trigger wave 1 (Beeliners). Let it complete or press F again.
6. Press F to trigger wave 2 (Snipers — 2 should spawn).
7. Observe Sniper behavior — verify ALL of these:
   a. Snipers approach from spawn, then STOP at a visible distance (they should NOT charge to melee like Beeliners)
   b. STATE label on Snipers shows SEEKING -> FIGHTING transition
   c. After entering FIGHTING, Snipers pause briefly (~1 second aim-up) before firing
   d. Sniper bullets are visibly SLOWER than Beeliner bullets
   e. Sniper bullets deal noticeably MORE damage per hit (check player health drain)
8. Fly your ship TOWARD a Sniper — close the distance aggressively:
   a. Sniper should transition to FLEEING state (visible in STATE label)
   b. Sniper should move AWAY from you
   c. Once you back off far enough, Sniper should return to SEEKING
9. Kill the Snipers — verify:
   a. Loot drops appear (coins and/or ammo)
   b. Wave completes — WaveManager is ready for the next F press
10. If any behavior is wrong, describe what you see.
</how-to-verify>
