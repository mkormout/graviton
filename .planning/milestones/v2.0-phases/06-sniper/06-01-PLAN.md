# Plan 06-01: Sniper Script + Bullet Scene

## Goal
Create the Sniper AI script with three-band distance management, aim-up telegraph, and FLEEING behavior, plus the sniper-bullet scene with higher damage/slower speed than the Beeliner bullet.

## Context
Phase 6 adds the second concrete EnemyShip type. The Beeliner (Phase 5) proved the full spawn-detect-fight-die-loot pipeline. The Sniper extends that with standoff distance management (three range bands + reverse thrust), a two-timer fire pattern (fire timer triggers aim timer, aim timer triggers shot), and the FLEEING state. All infrastructure is in place from Phases 4-5 — this plan creates the Sniper-specific code and bullet asset.

<description>
Create `components/sniper.gd` — the Sniper enemy type script. Model directly on `beeliner.gd` but with these additions:

**Class definition:**
- `class_name Sniper` / `extends EnemyShip`

**@export vars (per D-01, D-10, Claude's discretion for values):**
- `fight_range: float = 900.0` — outside this: approach (SEEKING)
- `comfort_range: float = 600.0` — inside this: reverse thrust (within-state behavior per D-07)
- `flee_range: float = 300.0` — inside this: trigger FLEEING (D-06)
- `safe_range: float = 700.0` — outside this during FLEEING: return to SEEKING (D-08)
- `aim_up_time: float = 1.0` — telegraph pause before shot (D-09, D-10)
- `bullet_speed: float = 1500.0` — slower than Beeliner's 4400 (D-13)

**Constants:**
- `FIGHTING_THRUST_MULT := 0.3` — reduced thrust in FIGHTING (D-03)

**Instance vars:**
- `_target: Node2D = null`
- `_bullet_scene := preload("res://prefabs/enemies/sniper/sniper-bullet.tscn")`

**@onready vars:**
- `_fire_timer: Timer = $FireTimer`
- `_aim_timer: Timer = $AimTimer` — one_shot = true in scene
- `_ammo_dropper: ItemDropper = $AmmoDropper`

**_ready():**
- Call `super()`
- Apply +-20% random variance to `thrust` and `max_speed` (same as Beeliner)
- Connect `_fire_timer.timeout` -> `_on_fire_timer_timeout`
- Connect `_aim_timer.timeout` -> `_on_aim_timer_timeout`

**_on_detection_area_body_entered(body):**
- Guard: `if dying: return`
- If `body is PlayerShip and current_state == State.IDLING`: set `_target = body`, `_change_state(State.SEEKING)`

**_tick_state(_delta) — per D-01 through D-08:**

CRITICAL: Check innermost range first (flee_range), then comfort_range, then fight_range. See Research Pitfall 4.

State.SEEKING:
- `look_at(_target.global_position)`
- if `dist < flee_range`: `_change_state(State.FLEEING)` (D-06 applies across states)
- elif `dist < comfort_range`: reverse thrust — `apply_central_force(away * thrust)` — no state change (D-07)
- elif `dist < fight_range`: `_change_state(State.FIGHTING)` — sweet spot (D-05)
- else: `steer_toward(_target.global_position)` — approach

State.FIGHTING:
- `look_at(_target.global_position)`
- if `dist < flee_range`: `_change_state(State.FLEEING)` (D-06)
- elif `dist < comfort_range`: reverse thrust at reduced power — `apply_central_force(away * thrust * FIGHTING_THRUST_MULT)` (D-03, D-07)
- else: gentle corrective thrust toward player — `apply_central_force(toward * thrust * FIGHTING_THRUST_MULT)` (D-03)

State.FLEEING:
- if `dist > safe_range`: `_change_state(State.SEEKING)` (D-08)
- else: full reverse thrust — `apply_central_force(away * thrust)` — also `look_at(_target.global_position)` to keep facing player

**_enter_state(new_state) — per D-09:**
- State.FIGHTING: `_fire_timer.start()` — begin fire cycle
- State.FLEEING: `_fire_timer.stop()` AND `_aim_timer.stop()` — cancel any in-progress aim-up (Research Pitfall 1)

**_exit_state(old_state):**
- State.FIGHTING: `_fire_timer.stop()` AND `_aim_timer.stop()`

**_on_fire_timer_timeout() — per D-09:**
- Guard: `if dying or current_state != State.FIGHTING: return`
- Start aim-up: `_aim_timer.start(aim_up_time)` — does NOT fire immediately (D-09)

**_on_aim_timer_timeout() — per D-09:**
- Guard: `if dying or current_state != State.FIGHTING: return`
- Call `_fire()`

**_fire() — per D-11, D-14:**
- Guard: `if dying: return`
- Instantiate `_bullet_scene` as `RigidBody2D`
- `fire_dir := Vector2.from_angle(global_rotation)` — single shot, no spread (D-11)
- Set `bullet.rotation = global_rotation`
- Set `bullet.linear_velocity = fire_dir * bullet_speed`
- If `spawn_parent`: `spawn_parent.add_child(bullet)`, position at `global_position + fire_dir * 350.0` (offset past HitBox radius 300 to avoid self-collision)
- Else: `push_warning("Sniper: spawn_parent not set")`
- Add print statement matching Beeliner pattern: `print("[Sniper] bullet spawned at %s vel=%s" % [bullet.global_position, bullet.linear_velocity])`

**die(delay) — following Beeliner pattern:**
- Guard: `if dying: return`
- Stop both timers: `_fire_timer.stop()`, `_aim_timer.stop()`
- `_ammo_dropper.drop()`
- `super(delay)` — triggers Body.die() which calls item_dropper.drop() for coins

Add `print("[Sniper] _enter_state: %s" % State.keys()[new_state])` in `_enter_state` for debugging (matches Beeliner convention).
</description>
