<!-- Muzzle flash pattern from RESEARCH.md Pattern 6: -->
```gdscript
# In weapon scene: CPUParticles2D named MuzzleFlash under Barrel
# one_shot = true, lifetime = 0.15, amount = 8, explosiveness = 0.9

# In weapon script:
@export var muzzle_flash: CPUParticles2D

func fire():
    if not can_shoot():
        return
    if muzzle_flash:
        muzzle_flash.restart()  # CPUParticles2D.restart() resets and re-emits
    super()
```

CPUParticles2D.restart() verified from Godot docs (Context7) — resets lifetime timer and re-emits all particles.
