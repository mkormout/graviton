## Deviations from Plan

### Auto-fixed Issues

**1. Pre-existing bug: bullets not spawning after inventory weapon remount**
- **Found during:** Smoke test (Task 2)
- **Issue:** `_slot_item_adding` in `mount-point.gd` instantiated a new weapon body but never set `spawn_parent`. The bullet was created in `fire()` but never added to the scene tree — silently discarded. Sound and ammo decrement still fired.
- **Fix:** Added `body_self._propagate_spawn_parent(body)` before `plug()` in `_slot_item_adding`
- **Files modified:** `components/mount-point.gd`
- **Verification:** Smoke test confirmed bullets fire correctly after unmount/remount
- **Committed in:** `5602e92`
