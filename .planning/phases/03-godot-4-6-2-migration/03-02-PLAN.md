The user must perform these steps in the Godot 4.6.2 editor (Claude cannot operate a GUI editor):

**Step 1 — Install Godot 4.6.2:**
- Download from https://godotengine.org/download/archive/4.6.2-stable/
- On macOS: extract the `.app` bundle alongside the existing 4.2.1 installation (they do not conflict)
- Do NOT delete Godot 4.2.1 yet — keep it until migration is verified

**Step 2 — Install 4.6.2 export templates:**
- Open Godot 4.6.2 editor (can use the Project Manager, or open a blank project)
- Go to Editor > Manage Export Templates > Download and Install
- Wait for the download to complete — templates are ~600MB
- These are required for MIG-03 (export verification in Task 2)

**Step 3 — Open the Graviton project in Godot 4.6.2:**
- In the Godot 4.6.2 Project Manager, Import the project at the repo path
- When prompted "This project was created in an older version of Godot", click "Open Anyway" / "Convert"
- The editor will update `project.godot` automatically (config/features will change from "4.2" to "4.6" or similar — per D-02, accept whatever Godot sets)

**Step 4 — Run UID upgrade (Godot 4.4+ requirement):**
- When the editor shows a "Upgrade UIDs" or "Update UIDs..." prompt, click "Upgrade" / "Accept"
- This generates `.gd.uid` files next to every `.gd` script (~29 files)
- If the prompt does not appear automatically, go to Project > Tools > Upgrade GDScript UIDs (or similar)

**Step 5 — Run "Upgrade Project Files":**
- Go to Project > Tools > Upgrade Project Files
- This re-saves all `.tscn` and `.tres` files with 4.6.2 format conventions in one pass
- Produces cleaner VCS diffs than letting the editor re-save files incrementally

**Step 6 — Check the Output panel:**
- After the project opens, check the Output panel (bottom of the editor) for any errors or warnings
- Expected: ZERO errors, ZERO deprecation warnings (Plan 01 already fixed the known ones)
- If any `auto_translate` warnings appear in scene files: open the scene in the editor, find the Control node with the warning, change its `auto_translate` property to use `auto_translate_mode` instead (Inspector panel)
- If any OTHER unexpected warnings or errors appear: note them — Claude will fix them in a follow-up if needed

**Step 7 — Commit the editor-generated changes:**
After the editor has finished converting, return to the terminal. Claude will run:
```bash
git add -A '*.uid' project.godot export_presets.cfg '*.tscn' '*.tres'
```
to stage all editor-modified files and new `.uid` files for commit.
