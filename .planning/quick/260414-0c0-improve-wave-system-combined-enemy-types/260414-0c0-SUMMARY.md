# Phase 260414-0c0 Plan 01: Improve Wave System — Combined Enemy Types Summary

**One-liner:** Multi-type wave spawning with 20 escalating definitions, 5-second auto-countdown between waves, and a centered 48px announcement label that fades out after 3 seconds.

## What Was Built

### Task 1: WaveManager multi-type waves with auto-countdown (`components/wave-manager.gd`)

- Added `countdown_tick(seconds_remaining: int)` signal
- Updated `wave_started` to `wave_started(wave_number: int, enemy_count: int, label_text: String)`
- New wave dict format: `{ "label": String, "groups": [{ "enemy_scene": PackedScene, "count": int }] }`
- Old `{ "enemy_scene": PackedScene, "count": int }` format still accepted (backwards-compat branch)
- Added `@export var countdown_seconds: float = 5.0`
- Countdown Timer created in `_ready()` as child node, 1-second interval, not one-shot
- After wave complete: if more waves remain, starts countdown from `countdown_seconds`; first wave requires manual F key trigger
- Threat T-0c0-02 preserved: `_enemies_alive = max(0, _enemies_alive - 1)`

### Task 2: 20 escalating waves + WaveHud update

**world.gd:**
- Replaced 10 single-type waves with 20 waves using new multi-group format
- Escalation arc: waves 1-3 single type, waves 4+ mix 2 types, waves 10+ mix 3 types, waves 19-20 all 5 types
- Wave 19 "Full Assault": 38 enemies total across all 5 types
- Wave 20 "Final Wave": 54 enemies total across all 5 types

**prefabs/ui/wave-hud.gd:**
- Updated `connect_to_wave_manager` to also connect `countdown_tick`
- `_on_wave_started` signature extended with `label_text: String`
- Announcement: sets label text, sets alpha=1.0, tweens to 0.0 over 3 seconds
- `_on_countdown_tick`: shows/hides `_countdown_label` with "Next wave in N..." text
- `_on_all_waves_complete`: hides countdown label

**prefabs/ui/wave-hud.tscn:**
- Added `CountdownLabel` inside `Panel/VBox` after `CountLabel` (font_size=13, hidden by default)
- Added `AnnouncementLabel` as direct CanvasLayer child (font_size=48, centered anchors, starts at alpha=0)
- Panel bottom offset extended to 92 to fit the new CountdownLabel
